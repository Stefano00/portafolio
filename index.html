Mi experiencia como programador comienza con mis estudios de electricidad cuando tenía 16 años programando circuitos para el control de motores y máquinas eléctricas. Siempre me gustaron los circuitos y la lógica que podía usar con estos, por esto mismo me titulé de ingeniero en Automatización con el fin de seguir programando microcontroladores, PLC's y sistemas SCADAS. Todo este proceso me llevó a estudiar programación web FULL STACK con java como lenguaje base, utilizando Spring Boot como tecnología para el desarrollo back-end y React-JS para el desarrollo Front-end. Con este conocimiento puedo mezclar la programación web de sistemas y la automatización de procesos. próximamente Con visión de integrarme en el mundo de la inteligencia artificial, Big Data y la ciencia de datos próximamente.
